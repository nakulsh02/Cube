# 3D Cube Project

A clean and interactive **3D Cube** created using web technologies. This project focuses on simplicity, smooth visuals, and a modern look.

## Features

- Interactive 3D cube
- Smooth rotations and animations
- Clean and minimal design
- Fully responsive (works on mobile and desktop)
- Runs directly in the browser
